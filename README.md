# 🐍 Snake Game - HTML5

A classic Snake game built with pure HTML5 Canvas, CSS3 and JavaScript.
Mobile-friendly with touch swipe controls and D-Pad buttons.

## 🎮 Features
- Mobile touch controls (swipe + D-pad)
- Keyboard support (Arrow keys)
- Score, Best Score & Level system
- Bonus food (⭐) for extra points
- Speed increases with level
- Neon dark theme

## 🚀 How to Play
- Use D-Pad buttons or swipe on screen
- Eat 🍎 to grow (+10 points)
- Eat ⭐ for bonus (+30 points)
- Don't hit walls or yourself!
